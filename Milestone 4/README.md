# Infosys Springboard

## Milestone 4: Dashboard and Report Export Module

---

## Project Overview

This project is developed as part of the **Infosys Springboard Individual Project**.

**Milestone 4** focuses on presenting the results generated in previous milestones through a **comprehensive and interactive dashboard**. The milestone enhances usability by visualizing skill gap analysis results and providing **downloadable PDF and CSV reports** using a Streamlit-based interface.

---

## Milestone 4 Objectives

* Build an interactive dashboard for skill gap analysis
* Display overall skill match statistics
* Visualize resume vs job description skill comparison
* Provide role-based views (Job Seeker and Recruiter)
* Highlight key skill match percentages
* Recommend upskilling areas
* Enable report export in PDF and CSV formats

---

## Features Implemented

* Interactive Skill Gap Analysis Dashboard
* Overall Match, Matched Skills, and Missing Skills summary cards
* Resume vs Job Requirement comparison using bar charts
* Key skill match percentage indicators
* Skill comparison using progress bars
* Role View with radar chart visualization
* Upskilling recommendations based on skill gaps
* Downloadable PDF report
* Downloadable CSV report

---
