# Infosys Springboard 

## Milestone 2: Skill Extraction using NLP

## Project Overview
This project is developed as part of the Infosys Springboard Individual Project.  
Milestone 2 focuses on extracting technical and soft skills from resumes and job descriptions using Natural Language Processing (NLP) techniques.

The goal of this milestone is to identify, classify, and visualize skills in a structured and interactive format.

## Milestone 2 Objectives
- Extract skills from resume and job description inputs
- Classify skills into Technical Skills and Soft Skills
- Apply NLP techniques for accurate skill identification
- Display extracted skills in a structured format
- Visualize skill distribution
- Provide filtering and refresh functionality

## Technologies Used
- Python 3
- Streamlit (for application interface)
- Natural Language Processing (NLP)
- spaCy
- BERT
- Plotly
- Custom CSS
- Git & GitHub
